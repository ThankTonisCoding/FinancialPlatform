# Trang web Bóng đá cơ bản

Đây là một website tĩnh đơn giản mô tả đội bóng và trận đấu để minh họa HTML/CSS/JS.

Các file chính:
- `index.html` - trang chính
- `css/styles.css` - stylesheet
- `js/app.js` - script để render dữ liệu mẫu

Hướng dẫn nhanh:
1. Mở `index.html` bằng trình duyệt (kéo thả hoặc chuột phải -> Open).
2. Tìm kiếm đội hoặc giải trên thanh tìm kiếm.
3. Bật/tắt chế độ tối bằng nút "Chế độ tối".

Ghi chú:
- Vì đây là một dự án tĩnh, bạn có thể mở trực tiếp file trong trình duyệt. Nếu bạn gặp vấn đề với `fetch` (nếu thêm sau này), hãy khởi chạy một server tĩnh (ví dụ `npx http-server` hoặc sử dụng Live Server trong VS Code).
